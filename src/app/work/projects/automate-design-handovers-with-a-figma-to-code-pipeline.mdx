## Overview

Built interactive Power BI dashboards to monitor access activity,
            compliance metrics, and provisioning trends.
## Highlights

-Converted Excel reports into Power BI dashboards
-Used SQL datasets for reporting pipelines
-Designed KPI and SLA visualizations
-Enabled interactive stakeholder reporting
## Technologies Used

- **Power BI**: For dashboard development and data visualization
- **SQL**: For data extraction and dataset preparation
- **DAX (Data Analysis Expressions)**: For KPI calculations and measures
- **Data Modeling**: For relationships and structured reporting datasets

## Challenges and Learnings

Faced difficulties in transforming inconsistent Excel and SQL data into a structured model while ensuring dashboard performance and meaningful KPI representation for non-technical stakeholders.
## Outcome

Delivered interactive Power BI dashboards that replaced manual reporting, improved visibility into compliance and access trends, and enabled faster, data-driven decision-making.
