## Overview

Designed and developed an interactive Power BI dashboard to provide real-time visibility into access activity, compliance status, and trends.

The solution replaced manual, Excel-based reporting with an automated and scalable analytics layer for improved governance and decision-making.


## Problem Statement

IAM and data analytics reporting was heavily dependent on static Excel files, leading to:

Limited visibility into access trends and compliance metrics
High manual effort in report preparation
Delayed insights for stakeholders and audit teams
No track of historical alerts


## Solution

Built an end-to-end reporting framework using Power BI, SQL, and DAX to centralize data and deliver interactive dashboards for stakeholders.

## Key Contributions
- Migrated legacy Excel reports into dynamic Power BI dashboards
- Integrated SQL-based datasets into reporting pipelines for structured analytics
- Designed KPI and SLA tracking metrics for compliance monitoring
- Developed interactive filters and views for stakeholder-specific insights
- Implemented data models to ensure performance and scalability

## Technologies Used

- **Power BI**: Dashboard development and visualization
- **SQL (Snowflake)**: Data extraction and transformation
- **DAX**: KPI calculations and business logic
- **Data Modeling**: Relationships and optimized reporting structure

## Challenges

Handled inconsistent and unstructured data across Excel and SQL sources. Focused on building a clean data model while ensuring dashboard performance and usability for non-technical stakeholders.

## Outcome & Impact

- Replaced manual reporting with automated dashboards
- Improved visibility into IAM compliance and access trends
- Enabled faster, data-driven decision-making for operations and audit teams
- Reduced reporting dependency on manual Excel workflows
