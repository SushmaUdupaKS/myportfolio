## Overview

Developed Python automation and SQL workflows to perform privileged
            access reviews and generate audit-ready Excel reports.
## Process Flow

- Extracted login and access data securely from Snowflake
- Executed optimized SQL queries for audit metrics
- Processed datasets using Pandas
- Used snowflake-connector-python for secure connectivity

## Technologies Used

- **Python**: For automation, data processing, workflow scripting
- **Pandas**: For data cleaning, transformation, and analysis
- **SQL**: For data querying and audit metric extraction
- **Snowflake**: For cloud data warehouse for secure data retrieval
- **snowflake-connector-python**: For secure database connectivity
- **Excel (OpenPyXL / XlsxWriter)**: For audit-ready report generation

## Challenges and Learnings

- Handling large datasets extracted from Snowflake efficiently without performance issues
- Ensuring secure database connectivity and credential management
- Cleaning inconsistent access data formats before analysis
- Optimizing SQL queries to reduce execution time and improve data retrieval speed

## Outcome

- Automated privileged access review process, reducing manual effort by 60% and improving reporting efficiency
- Generated consistent, audit-ready Excel reports with standardized data validation
- Enabled faster audit support through reliable and repeatable workflows.
