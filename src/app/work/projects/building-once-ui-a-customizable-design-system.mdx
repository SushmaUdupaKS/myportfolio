## Overview

Built an end-to-end automation workflow using Python and SQL to modernize privileged access reviews and generate audit-ready Excel reports. The solution reduced manual effort by 60%, replacing time-consuming manual processes with a scalable and reliable data pipeline that improved efficiency, consistency, and audit readiness.

## Process Flow

- Securely extracted login and access data from Snowflake
- Executed optimized SQL queries to compute audit and compliance metrics
- Processed and transformed datasets using Python (Pandas)
- Established secure database connectivity using snowflake-connector-python
- Generated standardized Excel reports for audit and compliance teams

## Technologies Used

- Python — Automation, workflow scripting, and data processing
- Pandas — Data cleaning, transformation, and analysis
- SQL (Snowflake) — Data extraction and audit metric computation
- Snowflake — Cloud data warehouse for secure data storage and retrieval
- snowflake-connector-python — Secure database connectivity
- Excel (OpenPyXL / XlsxWriter) — Automated audit-ready report generation

## Challenges and Learnings

- Efficiently handling large-scale datasets extracted from Snowflake
- Ensuring secure credential management and database connectivity
- Cleaning inconsistent and unstructured access data formats
- Optimizing SQL queries to improve execution speed and performance

## Outcome

- Reduced manual effort in privileged access reviews by 60%
- Automated generation of consistent, audit-ready Excel reports with validation checks
- Improved audit readiness through faster, repeatable, and reliable workflows
